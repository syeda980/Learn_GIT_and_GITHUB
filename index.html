What is Git?

So far the problem is considered one should need a platform where all the changes can be tracked. Here comes the savior
Git is a DevOps used for source code management. It is one of the most prominent version control systems (VCS) today that is widely used to handle small and large projects efficiently.Version control systems are software tools that help software teams manage changes to source code over time. It helps in tracking changes in source code, enabling different people to collaborate on different parts of the same program.

Git is easy to learn and has a tiny footprint with lightning fast performance. It outclasses SCM tools like Subversion, CVS, Perforce, and ClearCase with features like cheap local branching, convenient staging areas, and multiple workflows.

What is GitHub?

GitHub is an online software development platform used for storing, tracking, and collaborating on software projects. It enables developers to upload their own code files and to collaborate with fellow developers on open-source projects. GitHub also serves as a social networking site in which developers can openly network, collaborate, and pitch their work.

  1. git init

    git init is used to create new Repository. It can be used to create a empty repository or convert an existing, unversioned project to a Git repository. This is typically the first command you'll perform in a new project.

    This will create a folder named .git inside your project folder.It contains all of the necessary Git metadata for the new repository. This metadata includes subdirectories for objects, refs, and template files Since this is a hidden folder, Incase it is not visible, Go to view option from top menu in your explorere and check mark the hidden items option. After that the folder will be visible.
